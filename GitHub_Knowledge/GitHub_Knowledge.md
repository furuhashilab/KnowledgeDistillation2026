# Repository Overview

本リポジトリでは、MapSwipeを用いた地理空間情報分析や災害支援型マッピング活動に関する内容が整理されている。

MapSwipeは、衛星画像を多数の参加者が分散的に確認し、建物・道路・被災状況などを判定するクラウドソーシング型マッピングツールである。

OpenStreetMapと連携しながら、人道支援や災害対応を目的とした地図作成活動に活用されている。

# Knowledge Extracted

## 1. クラウドソーシング型GISの有効性

多数の参加者が小規模タスクを分担することで、大規模な衛星画像解析を高速化できる。

## 2. 災害対応における地理情報の重要性

被災地域では迅速な地図更新が必要となるため、OpenStreetMapとMapSwipeの連携が有効である。

## 3. 市民参加型マッピングの価値

専門家以外も参加可能な仕組みにより、大量の地理空間データ整備を実現できる。

## 4. OSS型GIS技術の活用

オープンソース地理空間技術を利用することで、低コストかつ拡張性の高い災害支援活動が可能となる。
