# fc_SpatialDesign

## 概要
古橋研究室の2020年度空間デザイン部の活動管理リポジトリである。descriptionに「2020年度空間デザイン部レポジトリ」と明記されており、デザイン理論（デザイン四原則）の学習、色彩設計、スライド制作、Markdown記法の習得など、デザインの基礎から応用までを扱う部活動の記録として機能している。AtsukoWakamatsu・Ayaka0324・rensanrenren・hharumax12など複数メンバーが参加している。

## 主な活動・成果
- Issue #1（Markdown記法）: AtsukoWakamatsuがMarkdown記法の学習・整理を担当。コメント1件あり（2020年12月）。
- Issue #2（色まとめ GitHub 1/18）: Ayaka0324・yukasaito11328が色彩に関する調査・まとめをGitHub上で実施（締切1/18）。
- Issue #4（スライドまとめ 1/25）: hharumax12・rensanrenrenが発表用スライドを担当（締切1/25）。
- Issue #5（デザイン四原則）: AtsukoWakamatsuがデザイン四原則（近接・整列・反復・コントラスト）について調査・まとめ。コメント1件あり。
- Issue #7（案出し）: AtsukoWakamatsuが企画・アイデア出しを担当。コメント4件と活発な議論があった（1月）。
- Issue #8・9・10（medium・スライド・発表）: rensanrenrenがMedium記事執筆・スライド・発表を担当（1月）。
- Issue #11（マニュアル書く）: AtsukoWakamatsuがマニュアル作成を担当。コメント1件あり。

## 使用技術・ツール
- Markdown（ドキュメント記法）
- GitHub Issues（タスク管理・作業進行）
- Medium（成果アウトプット媒体）
- デザイン四原則（近接・整列・反復・コントラスト）

## 得られた知見
- デザイン四原則（近接・整列・反復・コントラスト）を学習タスクとしてIssue化し、GitHubで管理することで、デザイン理論の習得をプロジェクト型学習として実施できる。色彩まとめ・スライド制作・マニュアル作成・Medium記事という一連のアウトプットフローは、デザイン部活動の標準的なサイクルとして参照可能。
